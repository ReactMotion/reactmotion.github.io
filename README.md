# ReactMotion Project Page

Academic project page for **ReactMotion: Generating Reactive Listener Motions from Speaker Utterance**.

## 🚀 Deploy to GitHub Pages

### Option 1: Deploy from root (recommended)

1. Push this repository to GitHub
2. Go to **Settings** → **Pages**
3. Under **Source**, select **Deploy from a branch**
4. Select **main** branch and **/ (root)** folder
5. Save — your page will be live at `https://<username>.github.io/ReactMotion/`

### Option 2: Deploy from docs folder

1. Create a `docs` folder and move `index.html`, `styles.css`, `script.js` into it
2. Or configure GitHub Pages to use the `docs` folder

## 📹 Adding Videos

1. Create a `videos` folder in the project root
2. Add your MP4 files (e.g., `demo1.mp4`, `demo2.mp4`)
3. Update the `src` attributes in `index.html` if you use different filenames

## ✏️ Customization

- **GitHub link**: Update the `href` in the GitHub button (line ~60 in index.html)
- **Paper link**: Update the "Read Paper" button URL when available
- **Author links**: Add Scholar/Homepage URLs for each author
